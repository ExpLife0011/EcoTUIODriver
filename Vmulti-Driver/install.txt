Prerequisites

Windows Driver Kit (https://www.microsoft.com/whdc/devtools/wdk/wdkpkg.mspx)

Building

Start a WDK build environment
1.Navigate to the location of your vmulti source root directory (eg. C:\projects\vmulti)
2.Build the drivers and test program using build -wgc
Installing
Note: This will not work on 64-bit systems without first properly signing the driver
3.Gather together the vmulti.sys/vmulti.inf/hidkmdf.sys files into single directory
4.Also copy the file WdfCoInstaller01009.dll (from the WDK) to your installation directory
5.Also copy the file devcon.exe (from the WDK) to your installation directory
Run the command: devcon install vmulti.inf djpnewton\vmulti
